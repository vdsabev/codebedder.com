Why?

CodeBedder is most suitable for light editing of content that was already written elsewhere - code samples or templates loaded from a database.

Lightweight

1.39KB gzipped. Syntax highlighting with a library of your choice like PrismJS.

Use anywhere

<code-bedder> web component - easy to integrate with any website.

Highlight anything

Specify any language supported by syntax highlighting libraries like PrismJS.

Why not?

This is a really basic code editor - a transparent textarea on top of a code block with syntax highlighting (grug-approved). You can't change indentation with Tab/Shift+Tab, or automatically insert spaces when you hit Enter.

If you need to do that consider using a more full-featured code editor.

Want to add this to your website?

Download codebedder.js, prism.js, and prism.css, or create a custom build of PrismJS to support languages other than HTML/CSS/JavaScript. For an HTML editor include the assets as follows: 

<html>
  <head>
    <link href="prism.css" rel="stylesheet" />
  </head>
  <body>
    <code-bedder lang="html" onhighlight="Prism.highlightElement(this)">YOUR_ESCAPED_HTML_CODE_HERE</code-bedder>
    <script src="prism.js"></script>
    <script src="codebedder.js"></script>
  </body>
</html>
